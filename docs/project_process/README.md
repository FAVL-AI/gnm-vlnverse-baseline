# FleetSafe VLN Reproducible Project Process

This folder contains the first-person reproducible project-process manuscript for FleetSafe Safe Visual-Language Navigation.

It documents the end-to-end build narrative, debugging log, command runbook, issue registry, verification evidence, mathematical safety explanation, and artifact map for the RTX desktop, Jetson robot, Yahboom M3Pro real robot, ROS 2 Humble stack, voice/text/image-conditioned VLN controller, Control Barrier Function safety filter, certificate logging, Isaac Sim/digital-twin plan, and nominal backbone models including GNM, ViNT, and NoMaD.
