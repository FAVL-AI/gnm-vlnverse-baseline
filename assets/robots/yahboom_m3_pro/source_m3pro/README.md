## Creating the URDF

See `urdf/PLACEHOLDER.md` for the complete creation guide.

Short version:

1. Obtain CAD from Yahboom (RosMaster M3Pro)
2. Export as URDF with 4 continuous mecanum wheel joints
3. Verify inertials against physical hardware (weigh components)
4. Place at `fleet_safe_vla/robots/yahboom/m3pro/urdf/yahboom_m3pro.urdf`
5. Run Stage 0: `./scripts/isaaclab/train_yahboom.sh --stage 0`

Do NOT create a fake URDF with made-up geometry. Incorrect inertials will
cause the sim-to-real gap to be uncharacterised, making Stage 5 transfer
unreliable.
